(260105) HTML 퀴즈 정답

Q1. 웹 브라우저의 렌더링 엔진에 대한 설명으로 옳은 것은? 정답: Chrome과 Edge는 현재 같은 엔진(Blink)을 사용한다
Q2. 크로스 브라우징(Cross-browsing)이란? 정답: 여러 브라우저에서 웹사이트가 제대로 작동하도록 하는 것
Q3. HTML, CSS, JavaScript의 역할이 올바르게 짝지어진 것은? 정답: HTML - 구조, CSS - 스타일, JavaScript - 동작
Q4. 웹 개발에서 "관심사의 분리(Separation of Concerns)"가 중요한 이유가 아닌 것은? 정답: 파일 크기가 작아진다
Q5. React에서도 HTML/CSS를 알아야 하는 이유는? 정답: React JSX는 결국 HTML로 변환된다
Q6. 브라우저에서 F12를 눌렀을 때 볼 수 있는 것이 아닌 것은? 정답: 서버의 데이터베이스 정보
Q7. 다음 중 올바른 파일 구조는? 정답: Option 2와 3 모두 옳다
Q8. HTML 파일의 기본 구조에서 빠진 것은? 정답: head
Q9. <!DOCTYPE html>에 대한 설명으로 옳은 것은? 정답: HTML5 문서임을 선언한다
Q10. 다음 코드의 실행 결과는? 정답: 안녕하세요 반갑습니다!
Q11. <meta charset="UTF-8">를 지우면 어떤 문제가 발생할 수 있나? 정답: 한글이 깨질 수 있다
Q12. 주석에 대한 설명으로 틀린 것은? 정답: 페이지 소스 보기에서도 볼 수 없다
Q13. Emmet에서 div>h2+p*3의 결과는? 정답: div 태그 안에 h2 1개, p 3개
Q14. strong 태그와 b 태그의 가장 큰 차이는? 정답: 의미의 유무
Q15. 다음 중 SEO에 가장 도움이 되는 태그는? 정답: strong 태그
Q16. 다음 중 가장 좋은 코드는? 정답: 2번 (article, h2, p를 사용한 코드)
