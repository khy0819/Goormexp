JavaScript 비동기 처리 예제

async function fetchData() {
  try {
    const response = await fetch('api/data');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error:', error);
  }
}

자주 묻는 질문 (FAQ)

Q. 프로그래밍을 전혀 해본 적이 없는데 수강 가능한가요?

A. 네, 가능합니다! 우리 강의는 완전 초보자도 쉽게 따라올 수 있도록 설계되었습니다. HTML 완벽 가이드부터 차근차근 시작하시면 됩니다.


Q. 강의는 어떤 방식으로 진행되나요?

A. 동영상 강의 + 실습 과제 + 퀴즈로 구성되어 있습니다. 각 세션마다 실습 코드를 제공하며, 궁금한 점은 커뮤니티에서 질문할 수 있습니다.


Q. 수강 기간 제한이 있나요?

A. 평생 수강이 가능합니다. 한 번 구매하시면 언제든지 복습할 수 있으며, 강의 업데이트도 무료로 제공됩니다.


Q. 수료증을 발급받을 수 있나요?

A. 각 강의를 80% 이상 수강하고 최종 프로젝트를 제출하시면 수료증을 발급해 드립니다. (PDF 파일)


Q. 환불 정책은 어떻게 되나요?

A. 수강 시작 후 7일 이내, 강의 진도 10% 미만인 경우 100% 환불 전액 환불이 가능합니다. (단, 결제 수수료는 제외)
