✅ (260107) HTML 퀴즈 정답 및 해설

Q1. 다음 중 a 태그의 필수 속성은?
정답: href
하이퍼텍스트 참조를 의미하며, 이동할 주소를 지정합니다.
Q2. 새 창으로 링크를 열고 싶을 때 사용하는 속성과 값은?
정답: target="_blank"
현재 창이 아닌 새 탭에서 페이지를 엽니다.
Q3. 보안을 위해 target="_blank"와 함께 사용하는 속성은?
정답: rel="noopener noreferrer"
부모 페이지의 정보를 보호하고 보안 취약점을 방어합니다.
Q7. label과 input을 올바르게 연결하는 방법은?
정답: label의 for 속성과 input의 id 속성을 같게
웹 접근성을 높여주며 클릭 영역을 확장해 줍니다.
Q15. charset 선언의 올바른 위치는?
정답: head 태그 내 가장 상단
브라우저가 텍스트를 읽기 전 인코딩 형식을 먼저 파악해야 글자 깨짐이 없습니다.
