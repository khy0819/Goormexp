✅ (260115) CSS 심화 퀴즈 정답 (1~15)

Q1. href가 특정 주소와 정확히 일치하는 링크 선택자는? 정답: a[href="https://www.google.com"]
해설: [attr="value"]는 속성값이 지정된 문자열과 완전히 일치할 때 선택합니다.
Q2. 클래스 이름에 "item"이 포함된 모든 요소 선택자는? 정답: [class*="item"]
해설: *=는 속성값 내에 해당 문자열이 포함되어 있으면 위치에 상관없이 선택합니다.
Q3. href가 "tel:"로 시작하는 링크 선택자는? 정답: a[href^="tel:"]
해설: ^=는 속성값이 특정 문자열로 시작하는 요소를 선택합니다.
Q4. .pdf 파일 링크만 선택하는 올바른 선택자는? 정답: a[href$=".pdf"]
해설: $=는 속성값이 특정 문자열로 끝나는 요소를 선택합니다.
Q5. :focus-visible의 설명으로 올바른 것은? 정답: 키보드로 포커스했을 때만 적용됨
해설: 마우스 클릭 시에는 포커스 테두리를 숨기고, 키보드 탭 키 사용 시에만 시각적 표시를 해줍니다.
Q6. li:not(:first-child) 코드의 결과는? 정답: 첫 번째 li를 제외한 모든 li에 border-top 적용
해설: :not()은 괄호 안의 선택자를 제외한 모든 요소를 선택합니다. 리스트 사이 구분선을 넣을 때 유용합니다.
Q7. :checked 가상 클래스를 사용할 수 있는 input 타입은? 정답: checkbox, radio
해설: 사용자가 선택(체크)한 상태를 감지하여 스타일을 바꿀 때 사용합니다.
Q8. article:has(img) 코드의 의미는? 정답: img를 포함한 article에 테두리 추가
해설: :has()는 특정 자식 요소를 가지고 있는 부모 요소를 선택하는 강력한 선택자입니다.
Q9. ::before와 ::after 사용 시 반드시 필요한 속성은? 정답: content
해설: 가상 요소는 content 속성이 정의되어야 브라우저에 표시됩니다. 값이 없어도 빈 문자열("")이라도 넣어야 합니다.
Q10. 다음 중 상속되지 않는 속성은? 정답: border
해설: color, font-size 등 글자 관련 속성은 상속되지만, border, margin, padding 등 박스 모델 속성은 상속되지 않습니다.
Q11. child 요소에 border: inherit; 적용 결과는? 정답: child div에 빨간 border가 표시된다.
해설: inherit 값은 해당 속성이 상속되지 않는 속성이더라도 부모의 값을 강제로 물려받게 합니다.
Q12. CSS Reset이 필요한 이유는? 정답: 브라우저마다 다른 기본 스타일을 통일하기 위해
해설: 브라우저 고유의 기본 여백(User Agent Stylesheet)을 제거하여 디자인의 일관성을 확보합니다.
Q13. 올바른 CSS 로드 순서는? (A:스타일, B:초기화) 정답: B → A (Normalize → Style)
해설: 초기화 파일을 먼저 로드한 뒤, 프로젝트 전용 스타일로 덮어씌우는 순서가 올바릅니다.
Q14. text-overflow: ellipsis 작동을 위해 필요한 속성은? 정답: white-space: nowrap;과 overflow: hidden; 둘 다 필요
해설: 텍스트가 줄바꿈되지 않고 영역을 넘쳐서 숨겨진 상태여야 말줄임표(...)가 나타납니다.
Q15. 3줄까지만 보이고 말줄임 처리를 하려면? 정답: -webkit-line-clamp: 3;
해설: 여러 줄 말줄임표를 구현하기 위해 사용하는 웹킷 전용 속성입니다.
