Q4. .pdf 파일 링크만 선택하려면? 정답: a[href$=".pdf"]
해설: $= 는 해당 값으로 '끝나는' 요소를 선택하는 특성 선택자입니다.
Q11. border: inherit; 적용 시 결과는? 정답: child div에 빨간 border가 표시된다.
해설: inherit 값은 부모의 속성값을 강제로 물려받게 합니다.
Q15. 3줄 말줄임표 처리를 위한 속성은? 정답: -webkit-line-clamp: 3;
해설: 범위를 넘는 텍스트를 숨기고 말줄임표를 찍을 줄 수를 결정합니다.
