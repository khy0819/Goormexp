(260106) HTML 퀴즈 정답지

Q1. HTML 속성에 대한 설명으로 올바른 것은? 정답: 여는 태그에만 작성하며 속성값은 따옴표로 감싼다
Q2. 전역 속성이 아닌 것은? 정답: src
Q3. 순서 없는 목록을 만드는 태그는? 정답: <ul>
Q4. ol 태그에서 로마 숫자로 표시하려면 어떤 속성을 사용해야 하는가? 정답: <ol type="I">
Q5. <ul>과 <ol> 태그의 직계 자식으로 올 수 있는 태그는? 정답: <li>만 가능
Q6. <ol start="5">의 의미는? 정답: 5번부터 시작하는 번호 매기기
Q7. img 태그의 필수 속성 2가지는? 정답: src와 alt
Q8. 상위 폴더로 이동하는 경로 표기는? 정답: ../
Q9. alt 속성이 중요한 이유가 아닌 것은? 정답: 이미지 파일 크기 최적화
Q10. 장식용 이미지(아무 의미 없는 꾸미기용 이미지)의 alt는 어떻게 작성해야 하는가? 정답: alt=""
Q11. 이미지 비율을 유지하면서 크기를 조절하려면? 정답: width 또는 height 중 하나만 지정
Q12. 표의 기본 구조를 만드는 필수 태그 3가지는? 정답: <table>, <tr>, <td>
Q13. 표에서 가로로 2개의 셀을 병합하려면 어떤 속성을 사용해야 하는가? 정답: colspan="2"
Q14. 표를 머리글, 본문, 바닥글로 구분하는 태그는? 정답: <thead>, <tbody>, <tfoot> 첫화면으로
