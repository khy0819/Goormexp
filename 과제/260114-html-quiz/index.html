✅ CSS 레이아웃 & 변수 퀴즈 정답

Q1. CSS 박스 모델의 4개 영역을 안쪽부터 순서대로 나열하면? 정답: Content → Padding → Border → Margin
해설: 실제 내용(Content)을 안쪽 여백(Padding)이 감싸고, 그 바깥에 테두리(Border), 마지막으로 요소 간의 간격(Margin)이 위치한다.
Q2. width: 200px, padding: 20px, border: 5px일 때 실제 너비는? 정답: 250px
해설: content-box 방식에서는 width(200) + 왼쪽/오른쪽 padding(40) + 왼쪽/오른쪽 border(10)를 모두 더해 250px이 된다.
Q7. background-size: cover의 특징은? 정답: 비율을 유지하며 요소를 꽉 채움
해설: 빈 공간이 생기지 않도록 배경을 꽉 채우는 방식이며, 이미지의 비율에 따라 상하 또는 좌우 일부가 잘릴 수 있다.
Q11. 이미지 위에 반투명 오버레이를 추가하는 방법은? 정답: 1번 (background-image에 linear-gradient와 url을 함께 사용)
해설: 배경 이미지 속성에 쉼표를 이용해 여러 값을 넣으면 먼저 선언된 순서대로 겹쳐진다. 그라데이션을 앞에 두어 오버레이를 만든다.
Q12. position: absolute 요소의 기준점은? 정답: position이 지정된 가장 가까운 부모
해설: static이 아닌(relative, absolute, fixed 등) 포지션을 가진 가장 가까운 조상 요소를 기준으로 위치가 결정된다.
Q13. 고정 헤더를 만들 때 사용하는 position은? 정답: fixed
해설: 스크롤을 내려도 항상 브라우저 화면(Viewport)의 특정 위치에 고정시키고 싶을 때 사용한다.
Q16. 요소를 화면에서 공간까지 완전히 제거하려면? 정답: display: none
해설: visibility: hidden이나 opacity: 0은 투명해질 뿐 공간을 차지하지만, display: none은 렌더링 트리에서 아예 제외된다.
Q19. Flexbox에서 양 끝 정렬 + 균등 간격을 설정하려면? 정답: justify-content: space-between
해설: 첫 번째 아이템은 시작점에, 마지막 아이템은 끝점에 붙이고 나머지 아이템 사이의 간격을 동일하게 배분한다.
Q22. CSS 변수를 전역으로 선언하는 선택자는? 정답: :root
해설: HTML 문서의 최상위 요소인 html을 가리키며, 이곳에 변수를 선언하면 문서 전체 어디서든 사용할 수 있다.
Q25. 부모 요소에 다른 색상 변수가 선언되었을 때 실행 결과는? 정답: A는 파란색, B는 빨간색
해설: CSS 변수는 상속되며, 동일한 변수명이라도 요소 자신이나 가장 가까운 조상에 선언된 값이 우선 적용된다.
