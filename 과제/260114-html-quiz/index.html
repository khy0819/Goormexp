✅ (260114) CSS 퀴즈 전체 정답 (1~25)

Q1. CSS 박스 모델의 4개 영역 순서는? 정답: Content → Padding → Border → Margin
해설: 안쪽(내용물)부터 바깥쪽(외부 간격)으로 확장되는 구조이다.
Q2. width: 200px, p: 20px, b: 5px일 때 실제 너비는? 정답: 250px
해설: 200 + (20*2) + (5*2) = 250px이다. (content-box 기준)
Q3. border-box 사용 시 width의 의미는? 정답: Content + Padding + Border의 너비
해설: 테두리까지를 전체 너비로 포함하여 계산을 직관적으로 만든다.
Q4. Padding과 Margin 설명 중 틀린 것은? 정답: Padding과 Margin 모두 클릭 영역을 확대한다
해설: 마진은 요소 외부 공간이므로 클릭 영역에 포함되지 않는다.
Q5. padding: 10px 20px 30px 40px;의 순서는? 정답: 상 우 하 좌
해설: 시계 방향(Top-Right-Bottom-Left) 순서로 적용된다.
Q6. Margin Collapsing(마진 상쇄)이란? 정답: 상하 margin이 만나면 큰 값만 적용된다
해설: 인접한 수직 마진이 하나로 합쳐지는 현상이다.
Q7. background-size: cover의 특징은? 정답: 비율을 유지하며 요소를 꽉 채움 (이미지 잘릴 수 있음)
해설: 요소에 빈틈이 생기지 않도록 배경을 가득 채우는 속성이다.
Q8. 히어로 이미지에 가장 적합한 속성값은? 정답: cover
해설: 큰 배경 이미지를 화면에 꽉 차게 보여줄 때 필수적이다.
Q9. 대형 배경 이미지의 반복을 막으려면? 정답: background-repeat: no-repeat
해설: 기본적으로 배경은 바둑판식으로 반복되므로 이를 명시적으로 꺼야 한다.
Q10. background-attachment: fixed의 효과는? 정답: 배경이 화면에 고정됨 (패럴랙스 효과)
해설: 스크롤 시 본문은 움직이지만 배경은 고정되어 깊이감을 준다.
Q11. 이미지 위 반투명 오버레이 추가 방법은? 정답: 1번 (linear-gradient와 url 혼합 사용)
해설: 쉼표를 구분자로 하여 다중 배경 레이어를 쌓을 수 있다.
Q12. position: absolute 요소의 기준점은? 정답: position이 지정된 가장 가까운 부모
해설: 기준이 될 부모에게 보통 relative를 부여하여 위치를 제어한다.
Q13. 고정 헤더를 만들 때 사용하는 position은? 정답: fixed
해설: 뷰포트(브라우저 창)를 기준으로 항상 고정된 위치를 유지한다.
Q14. 스크롤에 따라 특정 지점에서 고정되는 position은? 정답: sticky
해설: 일반적인 흐름을 따르다가 지정한 위치에 도달하면 고정된다.
Q15. z-index가 작동하지 않는 경우는? 정답: position: static
해설: 포지션이 기본값인 static 상태에서는 z-index가 무시된다.
Q16. 요소를 화면에서 공간까지 완전히 제거하려면? 정답: display: none
해설: 공간도 차지하지 않고 이벤트도 발생하지 않게 처리한다.
Q17. 요소를 감추되 클릭은 가능한 상태는? 정답: opacity: 0
해설: 투명해질 뿐 공간과 상호작용 기능은 그대로 유지된다.
Q18. Flexbox를 시작하는 속성은? 정답: display: flex
해설: 부모 요소에 선언하여 내부 자식들을 유연하게 배치한다.
Q19. 양 끝 정렬 + 균등 간격 설정은? 정답: justify-content: space-between
해설: 첫 아이템과 끝 아이템을 양 끝에 밀착시킨다.
Q20. Flexbox에서 간격을 주는 최신 방법은? 정답: gap
해설: 자식 요소 사이의 간격을 마진 없이 부모에서 일괄 조절한다.
Q21. 가로, 세로 완벽한 중앙 정렬 코드는? 정답: justify-content: center; align-items: center;
해설: 주축과 교차축 모두 중앙으로 정렬하는 가장 빠른 방법이다.
Q22. CSS 변수를 전역으로 선언할 때 사용하는 선택자는? 정답: :root
해설: 문서 전체에 적용되는 글로벌 스코프 변수를 선언할 때 쓴다.
Q23. 변수가 없을 때 기본값을 설정하는 방법은? 정답: color: var(--custom-color, blue);
해설: 쉼표를 사용하여 변수가 선언되지 않았을 때의 대체값을 지정한다.
Q24. @import로 CSS를 가져오는 올바른 방법은? 정답: @import url('common.css');
해설: CSS 파일 내에서 다른 CSS 파일을 모듈화하여 연결한다.
Q25. CSS 변수 범위 테스트 결과는? 정답: A는 파란색, B는 빨간색
해설: 변수는 선언된 요소와 그 자식들에게만 영향을 주는 스코프를 가진다.
